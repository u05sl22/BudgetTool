1. Overview
The Budget Calculator is a Java swing application that allows users to track multiple income sources
track muktiple spending categories
calculate total income, total expenses, and surplus/deficit
Supporting undo functionality to revert changes
Providing real-time spreadsheet-style updates

The application handles income and expenses entered as weekly, monthly or yearly amounts with automatic conversion


2. Features Implemented
   Income Management
   Three income fields: Wages, Loans, Other Income
   Period selection: Weekly, Monthly, Yearly for each field
   Automatic calculation Total income computed based on input and period

   Expense Management
   Three expense fields: Food, Rent, Other Expenses
   Period selection: Weekkly, Monthly, Yearly for each field
   Automatic calculation Total expenses computed based on input and period

   Financial Analysis
   Surplus/Deficit display**: Shows income minus expenses
   Visual indicators**: Black text for surplus/zero, red text for deficit
   Real-time updates**: Changes reflected immediately

   User Experience
   Spreadsheet behavior: Automatic updates on focus loss, Entry key, and period changes
   Multi-level undo: Stack-based undo system(10 levels maximum)
   Imput validation: Empty fields treated as 0, invalid inputs show error messages
